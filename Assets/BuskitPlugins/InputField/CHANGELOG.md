# Changelog

## [1.0.0] - 2020-03-17
### Added

* 添加兼容火狐、谷歌浏览器下能全屏输入中文的WebGL版InputField


## [1.0.1] - 2020-03-19
### Fixed

 * 添加了回车换行、回车提交事件
 * 全屏只支持火狐和谷歌浏览器

## [1.0.2] - 2020-04-03
### Fixed

 * 修复与Unity2019.X版本不兼容问题

## [1.0.3] - 2020-07-01
### Fixed

 * 修复InputField在切换场景后不销毁的情况下可能出现无法再次输入,并且键盘事件无响应的Bug

## [1.0.4] - 2020-08-03
### Fixed

 * 修复InputField的ReadOnly属性不生效Bug
